# TimeMoney - 時間金錢管理 

> 智慧型時間與金錢管理應用程式，幫助您有效管理時間和追蹤支出

## 功能特色 

- **智能任務管理** - 創建、排程和追蹤任務
- **支出追蹤** - 記錄和分析個人支出
- **數據可視化** - 圓餅圖和趨勢圖表
- **PWA 支持** - 可安裝到手機主螢幕
- **雲端同步** - 使用 Firebase 即時同步
- **現代 UI** - 深色主題設計

## 技術棧 

- **前端**: React 19 + TypeScript + Vite
- **樣式**: Tailwind CSS v4
- **後端**: Firebase (Firestore + Auth)
- **PWA**: Vite PWA Plugin
- **圖表**: Recharts
- **圖標**: Lucide React

## 本地開發 

```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 建置
npm run build

# 預覽建置結果
npm run preview
```

## PWA 安裝 

1. 在手機瀏覽器中打開應用
2. 點擊瀏覽器選單中的「加入主螢幕」
3. 享受原生應用般的體驗！
